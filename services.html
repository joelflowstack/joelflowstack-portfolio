Services

3D interactive websites

Three.js-driven sites where the visual is load-bearing, not decorative — scroll-driven scenes, WebGL heroes, product visualizers. Typically 2–4 weeks depending on scene complexity.

AI chatbots & agents

Assistants wired to real LLM APIs (Groq, OpenAI-compatible endpoints) with actual context and fallback handling — not a scripted FAQ bot. Usually 1–3 weeks including integration testing.

Discord / Telegram / WhatsApp bots

Community bots, support bots, and inquiry routers built directly on each platform's API — moderation, onboarding flows, and ticket routing included where relevant.

n8n workflow automation

Connecting the tools you already use — forms, CRMs, spreadsheets, notifications — so the manual steps between them disappear. Often the fastest win of the four.

Common questions

FAQ

How long does a typical project take?

Most single-service engagements (a chatbot, a bot, an automation) land in 1–3 weeks. 3D web builds with custom scenes run 2–4 weeks depending on complexity. Combined projects are scoped individually.

Do I need to know how to code or use git?

No. Every deliverable is built to deploy through the GitHub web UI and Vercel — no CLI, no local dev environment required on your end.

What happens after launch?

Ongoing support is normal, not an upsell. Copy changes, new integrations, and bug fixes are usually turned around within a day or two.

Can you work with an existing site or codebase?

Yes — plenty of projects are additions to something that already exists, not full rebuilds. Send what you have and it'll get scoped honestly.
