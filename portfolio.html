Flow V3

AI AgentGroqVercel

An AI chat agent originally built on a free-tier host, migrated to Groq's LPU inference for near-zero cold-start latency, running llama-3.3-70b-versatile with an 8B fallback model for reliability under load.

Embedded here as a real API-calling widget — not an iframe — POSTing directly to its own backend, with rate-limit handling and a 25-second timeout so it fails gracefully instead of hanging.

The migration

Free-tier cold starts meant multi-second delays before Flow could respond — enough to lose someone's attention entirely. Moving to Groq's inference stack cut that to near-instant, with a smaller fallback model kept in reserve for when the primary is under load.
